Secura (AR Door lock)

Secura is a Double password protedted door lock operated by an Android application based on augmented reality. There is no lock and key mechanism in this system, The application identifies the door based on the QR code generated by user and the keypad becomes visible only when it identifies the QR code. Later the user is promted to enter the four digit password on the screen to open the door. This unity allows the user to change both the lock (QR code ) and the password at any time and allows exchange of locks between doors
Technology : The first password mentioned here is the QR code generated by the user. It can be sticked onto any door powered by secura. The android application will generate the keypad only when it finds the marker ( QR Code ) and it matches the password with the data given by user. And instructions will be given through bluetooth to open the door
