UV Transilluminator

UV illuminator for aggar gel

UV transilluminator is a simple device which will let you see the DNA
in an agar gel. Transilluminator is a small glass box with a UV source
when the UV light falls on the gel , the dna will produce visible light spectrum
and the user will be able to see the dna bands
