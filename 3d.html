3-D Printer

Indigenous design and algorithme

Worked as a Team member for development of 3-D printer in Mechatronics lab NIT Calicut. We developed indigenous algorithm for motor control. The printer works from the data that is stored in an SD Card. The data is in the format of G Code. G code contains the necessary data about the coordinates to the printer  our algorithm reads the g code and controls the motors and the extruder according to it  The external structure were created in solidworks and implemented on aluminium channels. The printer was able to give a resolution near to 2mm
