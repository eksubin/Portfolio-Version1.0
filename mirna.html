miRNA DB

miRNA DB is a dedicated android application for micro RNA database and its one of the first android application dedicated for acessing micro RNA database we generated. I've designed it as Mini project for my undergraduate course at NIT Calicut Dept of biotechnology.
Technology : The android application works on amazon aws for database support. A SQL database is been hosted in amazon EC2 services and server side codes is been written to avoid unwanted acess of our database. The query ill be posted throught the android application and the domain user want to search for can be selected. The user interface is been designed to be very easy to use and fluid. For now the data base is been blocked for adding content to the database.
