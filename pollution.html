Pollution Controller for Vehicles

Pollution control system for two wheelers

This is a project that i have done in my school life. The problem was that there were no pollution controlling mechanism for 2 wheelers. As they contribute a lot of pollutants to the atmosphere, the need for a cheaper machine was important.  The pollution controller contains three subunits (a electrical precipitator, which precipitates carbons particles in the smoke,the precipitator works from the 12v battery of the vehicle. The second is a chemical unit which traps Carbon dioxide in it.The third component is a bacterial degradation unit which degrades harmful fluids in the exhaust gas)
