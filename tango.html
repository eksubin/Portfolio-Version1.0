Tango

Tango is a personal chating application i built based on sinch chat engine. Unlike other apps tango stores data in the server side not on the client side which gives more security for the data send through tango. Since tango is a basic framework it shows all the users in the main winsow. One of the main advantage of tango is that Tango automatically destroys all the chat history on exiting from the app ie all the chat history will be cleared from the server at the moment the user press back or exits from the androis app .Users are able to sign up easily with a new account and the details will be stored in a cloud server
